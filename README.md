# Simulateur salaire & retraite

Application web statique pour estimer :

- le passage salaire brut vers net ;
- le brut nécessaire pour atteindre un net cible ;
- une estimation pédagogique de retraite brute et nette.

Le simulateur est contenu dans `index.html` et peut être déployé directement sur Vercel.

## Déploiement Vercel

Importer ce dépôt GitHub dans Vercel avec les paramètres par défaut :

- Framework Preset : Other
- Build Command : vide
- Output Directory : `.`

Vercel servira automatiquement `index.html`.

## Note

Les résultats sont indicatifs et ne remplacent pas une fiche de paie officielle ni les simulateurs retraite officiels.
